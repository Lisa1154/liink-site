Waarom we websites en webshops met custom code schrijven

 26 juli '22  —    minute read

Een websitebouwer zoals het populaire WordPress wordt veel gebruikt. Waarom wij er dan toch voor kiezen om een website of webshop volledig met eigen code te schrijven en niet met een websitebouwer zoals WordPress? Dat heeft een aantal gunstige redenen, zowel voor onszelf als voor jou als klant.

Schone code

Websitebouwers zoals WordPress of Wix zijn heel makkelijk in gebruik, vooral wanneer je nog nooit eerder een website hebt gebouwd. Ideaal! Maar wat je aan de voorkant niet kunt zien is dat deze websitebouwers door hun ‘drag & drop’ methode vaak overbodige code bevatten. Hiervan is vaak minder dan 10% écht van belang voor de klant. Ook lijkt het ruime aanbod in al die plugins op het eerste gezicht erg handig, maar vaak worden deze gemaakt door hobbyisten, én gratis, waardoor ze slechte code kunnen bevatten.

Waarom schone code zo belangrijk is? Wanneer jouw nieuwe website net online is gaat Google deze afstruinen om de inhoud te beoordelen. Hoe schoner de code, hoe gemakkelijker het voor Google is om de inhoud van de website te beoordelen.

Omdat wij onze websites en webshops volledig in onze eigen code schrijven, gekoppeld aan Craft, is van overbodige code geen sprake.

Snelle laadtijd

Schone code zorgt daarnaast voor een snelle laadtijd van je website of webshop. Een trage website met een overkill aan plugins en slechte code heeft een sterk negatieve invloed op de ranking in zoekmachines. Een betere positie in Google is natuurlijk wat je wilt!

Easy onderhoud

Een ander voordeel van het schrijven met eigen code is dat, mocht er een probleem zijn ontstaan in de code, deze snel kan worden gevonden en opgelost. Je hoeft namelijk niet in andermans code te duiken, wat een hoop tijd scheelt.

Ook werken updates altijd, die zijn het eerste jaar zelfs helemaal gratis. Je hebt niet te maken met (veelal) in elkaar geknutselde plugins die na zo’n update vaak niet meer werken.

Maatwerk, maar dan écht

Custom code betekent ook: altijd een uniek design, want we werken niet met standaard thema’s. Daardoor kan er precies gebouwd worden wat er nodig is (zie ook punt 1). Je website of webshop is helemaal op maat in te richten en je kunt zelf heel eenvoudig aanpassingen doen.

Veiligheid gaat voor alles

Een website waarin grote hoeveelheden gratis plugins wordt gebruikt is kwetsbaar en kan daardoor aantrekkelijk zijn voor hackers. Aan Craft kunnen ook plugins gekoppeld worden. Deze zijn altijd gebouwd door professionals die veiligheid voorop stellen. Over het algemeen zit de meeste functionaliteit al standaard in het systeem. Lees hier meer over.
